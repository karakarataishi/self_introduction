# Quiz App

## 概要

自己紹介も兼ねて、4択クイズアプリを作成しました。
問題に答えながら、自分のことを知ってもらえるような内容になっています。

## 実行方法

1. zipフォルダをダウンロードして解凍する
2. フォルダのパスで仮想サーバーを起動する
3. `index.html` をブラウザで開く

## 工夫した点

* 正誤判定時に音が鳴るようにした
* 正誤判定時に画像を表示するようにした
* 現在何問目を解いているか分かるように、問題数を表示するようにした
* クイズデータを JSON ファイルに保存し、JavaScript で読み込むことで、問題の追加や編集をしやすい構成にした

## 使用技術

* HTML
* CSS
* JavaScript
* JSON

## ファイル構成

* `index.html` : タイトル画面
* `introduction.html` : ルール説明画面
* `quiz.html` : クイズ画面
* `result.html` : 結果表示画面
* `style.css` : スタイル
* `index.js` : JSONファイルからのデータ読み取り、およびクイズの正誤判定
* `result.js` : クイズ結果の表示
* `quiz.json` : クイズデータを格納
